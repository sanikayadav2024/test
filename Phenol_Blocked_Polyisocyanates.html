Applications

  • ULTRA CX STABLE-O : In combination with saturated polyester polyol, it can be used to formulate wire enamel coatings with very good thermal stability. It is also used in other coating formulations.

  • ULTRA AX STABLE-H : In combination with saturated polyester polyol, it can be used to formulate wire enamel coatings with improved solderability. It is also used in other polyurethane varnish formulations.

  • ULTRA AX STABLE : In combination with saturated polyester polyol, it can be used to formulate wire enamel coatings with improved solderability. It is also used in other polyurethane varnish formulations.

Special Applications

  • ULTRA CX STABLE-O : Specific formulations of wire enamels, made by using Ultra CX Stable-O, are found to be suitable for magnet wires with applications in EV’s (Electric Vehicles).
